Linguagens de Programação

COBOL

Ainda usado em sistemas bancários. Criada em 1959, continua sendo crítica para a infraestrutura financeira global.

C

Base para muitos sistemas operacionais. Criada por Dennis Ritchie em 1972, revolucionou a programação.

HTML

Não é linguagem de programação, mas estrutura a web. Criada por Tim Berners-Lee em 1989.

Spam vem de um presunto

O termo "spam" para emails indesejados vem de um esquete do Monty Python onde a palavra SPAM (marca de presunto enlatado) era repetida incessantemente.

Monty Python

Bugs Absurdos da História

Ariane 5 explodiu em 37 segundos

Em 1996, o foguete Ariane 5 explodiu por causa de uma conversão de número de 64 bits para 16 bits.

Custo do erro: aproximadamente 370 milhões de dólares.
1996

Y2K (Bug do Milênio)

Sistemas antigos usavam apenas dois dígitos para o ano. Em 2000, poderiam interpretar "00" como 1900.

O mundo gastou bilhões se preparando... e quase nada aconteceu.

2000

Bug do Pentium

Um erro microscópico na divisão de números em processadores Intel causava resultados errados em cálculos específicos.

Custo estimado: 475 milhões de dólares em recall.
